Une suite d'animation complète pour Gutenberg : GSAP, Rive, Spline 3D, Lottie, Matter.js et plus encore.

== Description ==

WP Simple GSAP Block transforme votre éditeur Gutenberg en un studio de motion design professionnel. Plus besoin de coder du Javascript complexe pour obtenir des effets dignes des "Site of the Day" Awwwards.

Fonctionnalités Principales :

GSAP ScrollTrigger : Animations liées au défilement (Fade, Slide, Scale).

Rive & Lottie : Support natif des formats d'animation vectorielle interactifs (.riv, .json).

Spline 3D : Intégration facile de scènes 3D interactives.

Physique (Matter.js) : Transformez vos éléments en objets soumis à la gravité.

Effets Cinématiques : Grain de film, Smooth Scroll (Lenis), Tilt 3D.

Typography FX : Texte dégradé, Scramble (Hacker), Vagues animées.

== Installation ==

Téléversez le dossier wp-simple-gsap dans le répertoire /wp-content/plugins/.

Activez l'extension via le menu 'Extensions' de WordPress.

Ouvrez une page ou un article et ajoutez le bloc "GSAP Dimension".

== Frequently Asked Questions ==

= Est-ce que cela ralentit mon site ? =
L'extension charge uniquement les scripts nécessaires (GSAP, Rive, etc.) sur le front-end. Nous utilisons les versions CDN optimisées pour garantir la performance.

= Puis-je utiliser mes propres fichiers Rive ? =
Oui, il suffit de coller l'URL de votre fichier .riv dans les réglages du bloc.
